球の中心が原点 ( 0, 0, 0 ) の球は以下の式で表すことができる。

$$
x^2 + y^2 + z^2 = r^2
$$

このとき、
* $r > 0$ : 球の半径

---

中心位置を ( cx, cy, cz ) とすると、

$$
( x - cx )^2 + ( y - cy )^2 + ( z - cz )^2 = r^2
$$

このとき、  
* $( cx, cy, cz )$ : 球の中心の座標  
* $r > 0$ : 球の半径
